# Experiments — Test Your Knowledge

> This folder is your sandbox. Use it to experiment, test ideas, and prove to yourself that you understand what you've learned.
> No rules here — just code, break things, and learn.

## How to Use This Folder

1. After finishing a day's lesson, come here and try to build something WITHOUT looking at the lesson file
2. If you get stuck, that's the gap in your knowledge — go back and review
3. Keep your experiments messy — this isn't for show, it's for learning

## Suggested Structure

```
experiments/
├── README.md                  # This file
├── week1_test.py              # Test yourself on Week 1 topics
├── week2_test.py              # Test yourself on Week 2 topics
├── mini_projects/             # Small projects combining multiple topics
├── challenges/                # Self-imposed coding challenges
└── scratch.py                 # Quick throwaway experiments
```

## Quick Challenges Per Week

### After Week 1–2 (Python Fundamentals)
- Build a calculator using only functions
- Create a contact book using dictionaries
- Write a number guessing game with loops

### After Week 3 (OOP)
- Build a simple bank account system with classes
- Create a to-do list app using OOP

### After Week 4 (Intermediate)
- Write 3 decorators (timer, logger, retry)
- Build a generator that reads a large file line by line

### After Week 5–6 (DSA)
- Implement a stack and queue from scratch
- Write binary search without looking at notes
- Solve 5 LeetCode easy problems in a row

### After Week 7 (Git, SQL, Tools)
- Create a Git repo, make branches, create a merge conflict, resolve it
- Write 5 SQL queries on a sample database

### After Week 8–9 (NumPy, Pandas, Viz)
- Load a CSV, clean it, analyze it, plot it — all from memory
- Recreate a Kaggle EDA notebook from scratch

### After Week 10 (Production)
- Build a REST API that serves predictions from a simple model
- Containerize it with Docker

## The Rule

> If you can't do it without looking at the lesson, you haven't learned it yet. Come back and try again tomorrow.
