## 📋 How to Use This Repo

1. **Follow the roadmap** — Start with `ROADMAP.md` (Phase 1), then `junior_ml_engineer_roadmap.md` (Phase 2)
2. **Follow the daily routine** — See `daily_routine_plan.md` for your daily schedule
3. **Do the exercises** — Each day file has solved exercises + unsolved practice exercises
4. **Test yourself** — Use the `experiments/` folder to code without looking at lessons
5. **Push daily** — Every day, commit and push your work to build your GitHub profile
