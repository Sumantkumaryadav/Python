## ⏰ Weekday Routine (Mon–Fri) — ~3 Hours

```
┌──────────────────────────────────────────────────────────┐
│  BLOCK 1: LEARN (45 min)                    🧠 Brain     │
│  ─────────────────────────────────────────────────────── │
│  • Watch course video / read chapter                     │
│  • Take handwritten or typed notes                       │
│  • Focus on ONE concept per session                      │
│                                                          │
│  BLOCK 2: CODE (60 min)                     💻 Hands     │
│  ─────────────────────────────────────────────────────── │
│  • Code what you just learned — NO copy-paste            │
│  • Do exercises from the lesson                          │
│  • Break things, experiment, debug                       │
│                                                          │
│  BLOCK 3: PROBLEM SOLVING (30 min)          🏋️ Muscle    │
│  ─────────────────────────────────────────────────────── │
│  • Phase 1 (Weeks 1–4): SKIP — focus on Python          │
│  • Phase 1 (Weeks 5–6): DSA exercises from lesson       │
│  • Phase 1 (Weeks 7–10): 1 LeetCode easy/day            │
│  • Phase 2 (Month 0–3): SKIP — focus on ML + math       │
│  • Phase 2 (Month 4+): 3–4 LeetCode problems/week       │
│  • If stuck >15 min, read the solution, then redo it     │
│                                                          │
│  BLOCK 4: READ & REFLECT (15 min)           📖 Growth    │
│  ─────────────────────────────────────────────────────── │
│  • Read 1 blog post / paper / documentation              │
│  • Write 2–3 sentences about what you learned today      │
│  • Push all code to GitHub                               │
│                                                          │
│  BLOCK 5: REVIEW (15 min)                   🔄 Retain    │
│  ─────────────────────────────────────────────────────── │
│  • Revisit yesterday's topic (spaced repetition)         │
│  • Quiz yourself: can you explain it without notes?      │
│  • Update your personal cheat sheet                      │
└──────────────────────────────────────────────────────────┘
                    TOTAL: ~2 hr 45 min
```

## 📅 Phase 1: Python + DSA Routine (Weeks 1–10)

### Weeks 1–2 (Python Fundamentals)
```
Block 1: Day lesson file (variables, strings, loops, etc.)
Block 2: Code exercises from the lesson — type everything yourself
Block 3: SKIP (no problem solving yet)
Block 4: Read Python docs / Real Python articles
Block 5: Explain yesterday's topic in your own words
Weekend: Mini-project using the week's concepts
```

### Week 3 (OOP)
```
Block 1: Classes, inheritance, dunder methods
Block 2: Build OOP exercises — Library Manager, Bank Account
Block 3: SKIP
Block 4: Read about OOP design patterns
Block 5: Draw class diagrams from memory
Weekend: OOP project
```

### Week 4 (Intermediate Python)
```
Block 1: Lambda, generators, decorators, context managers
Block 2: Write decorators, generators — experiment
Block 3: SKIP
Block 4: Read advanced Python blogs
Block 5: Quiz: can you write a decorator without looking?
Weekend: Refactor old code using new concepts
```

### Weeks 5–6 (DSA)
```
Block 1: DSA concept (Big-O, two pointers, trees, etc.)
Block 2: Implement algorithms from scratch in Python
Block 3: DSA exercises from the lesson + 1 LeetCode easy
Block 4: Read algorithm explanations (NeetCode, visualgo.net)
Block 5: Redo yesterday's problem without looking
Weekend: LeetCode practice day (5–10 easy problems)
```

### Weeks 7–10 (Tools, NumPy, Pandas, Production)
```
Block 1: Library/tool lesson (NumPy, Pandas, FastAPI, etc.)
Block 2: Hands-on coding with real datasets
Block 3: 1 LeetCode easy/day (keep DSA sharp)
Block 4: Read library documentation
Block 5: Review yesterday's code
Weekend: Data project or capstone work
```

## 📅 Phase 2: ML Engineer Routine (Months 0–12)

### Month 0 (Math Foundations)
```
Block 1: Khan Academy / 3Blue1Brown videos
Block 2: Solve math problems by hand + code them in Python
Block 3: SKIP (focus on math)
Block 4: Read StatQuest blog posts
Block 5: Quiz yourself on formulas
Weekend: Implement math concepts in NumPy
```

### Months 1–3 (ML Foundations)
```
Block 1: Andrew Ng's course / Géron's book
Block 2: Build models in Scikit-learn
Block 3: SKIP (focus on ML)
Block 4: Read Scikit-learn docs / ML blogs
Block 5: Explain algorithms in your own words
Weekend: Kaggle datasets + mini-projects
```

### Months 4–6 (Deep Learning + GenAI)
```
Block 1: fast.ai / PyTorch tutorials / HuggingFace course
Block 2: Build neural networks, fine-tune models
Block 3: CONTINUE LeetCode — 1 easy/medium problem/day
         (you already have DSA basics from Python Phase)
Block 4: Read papers (Illustrated Transformer, etc.)
Block 5: Review architectures — draw them from memory
Weekend: CV/NLP/RAG projects
```

### Months 7–9 (MLOps & Production)
```
Block 1: Chip Huyen's book / cloud docs / Airflow tutorials
Block 2: Build pipelines, deploy models, write APIs, set up monitoring
         (you covered FastAPI/Docker basics in Python Phase — go deeper now)
Block 3: LeetCode easy/medium mix
Block 4: Read MLOps blogs (Made With ML, InterviewNode, etc.)
Block 5: Practice ML system design questions out loud
Weekend: End-to-end deployment projects + design docs
```

### Months 10–12 (Portfolio & Job Prep)
```
Block 1: Interview prep materials + ML system design practice
Block 2: Polish portfolio projects (add monitoring, explainability)
Block 3: LeetCode medium + timed practice
Block 4: Research companies, read job descriptions
Block 5: Practice explaining projects to non-technical people
Weekend: Mock interviews + applications + blog writing
```

## 🧠 Productivity Rules

1. **Same time every day** — Build a habit, not motivation
2. **Phone on silent** — 3 focused hours > 6 distracted hours
3. **Code first, watch second** — If you can only do one block, do Block 2
4. **Never zero days** — Even 30 min on a bad day counts
5. **Teach what you learn** — Explain to a friend, write a blog, rubber duck it
6. **Track streaks** — Use a habit app (Habitica, Streaks, or a simple calendar)
7. **Rest is productive** — Take Sunday evening OFF completely
8. **Compare to yesterday-you, not Twitter-people** — Everyone's path is different
